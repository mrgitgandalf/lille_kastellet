# Lille Kastellet - lillekastellet.no

## Prosjekt
Enkel statisk hjemmeside for domenet lillekastellet.no.

## Teknologi
- Ren HTML5 og CSS (ingen byggverktøy eller rammeverk)
- Hosting: Domeneshop webhotell

## Filstruktur
```
index.html          # Hovedside
css/style.css       # Styling
images/             # Bilder
```

## Deploy
Last opp filene (index.html, css/, images/) til Domeneshop via FTP eller filbehandleren i kontrollpanelet.
